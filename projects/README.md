# Folder Portofolio Projects
# ============================
# Folder ini berisi hasil git clone dari project-project Laravel
# yang ditampilkan di halaman portofolio Moduvox.
#
# Cara penggunaan:
#   cd projects/hris
#   git clone https://github.com/IT-DP-TASPEN/<repo-hris>.git .
#
# Setiap subfolder = 1 project Laravel utuh.
# Pastikan setiap project sudah dikonfigurasi .env masing-masing.
#
# Daftar Project:
#   - hris/              → HRIS & Absensi (hris.moduvox.com)
#   - siardi/            → SIARDI Arsip Digital (siardi.moduvox.com)
#   - crm/               → CRM Solutions (crm.moduvox.com)
#   - core-banking/      → Core Banking Koperasi (core-banking.moduvox.com)
#   - bantuan-potong/    → Aplikasi Bantuan Potong (bantuan-potong.moduvox.com)
#   - sinergi/           → Sinergi Kolaborasi (sinergi.moduvox.com)
#   - btn-channeling/    → BTN Channeling (btn-channeling.moduvox.com)
#   - web-company-profile/ → Web Company Profile (wcp.moduvox.com)
